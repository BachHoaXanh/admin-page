# Install Docker & Docker Compose

```
$ docker-compose up
```

The Database container will create with the name **"admin-db"** & port **"3308"**

# Install & Run Server

```$xslt
$ cd server
```

```
$ npm install
-- or --
$ yarn install
```
Run with Production Mode
```$xslt
$ npm run start
-- or --
$ yarn start
```

Run with Develop Mode
```$xslt
$ npm run start:dev
-- or --
$ yarn start:dev
```
